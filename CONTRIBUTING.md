# Contributing to PandaLab

We appreciate your interest in contributing to PandaLab projects.

At this stage, contributions are managed privately to maintain project security and quality.  
External pull requests are not accepted directly on public repositories.

## How to Request to Contribute
If you wish to join or propose a contribution, please send an email to the organization with:
- Your name or alias
- Relevant skills or areas of contribution (design, frontend, backend, infrastructure, etc.)
- The project you are interested in

**Contact:** team@pandalab.io

Once reviewed, the team will respond with further information on how to collaborate.

## Internal Contributions
Members of the organization may update this repository and other PandaLab projects following internal review and branch protection rules.

All updates should follow:
- Conventional Commit messages
- Existing documentation standards
- Code review by another member before merging
